Getting Started with JQMDP

前提条件として jQuery mobile を利用する準備が整っていなければならない。
jQuery mobile の準備については こちらへ(http://jquerymobile.com)

まずは jquery.mobile.jqmdp-1.0.js をダウンロードして自分のWebサイトに配置する。
配置した jquery.mobile.jqmdp-1.0.js を jquery.mobile-1.0.js の直前に読み込めば準備完了である。

<script src="jquery-1.6.4.js"></script> <script src="jquery.mobile.jqmdp-1.0.js"></script> <script src="jquery.mobile-1.0.js"></script>

もっとも簡単な例を示す。 
<div>Date: <span data-dp-text="(new Date())"></span></div>
このコードは以下のように このページを表示した日時が表示される。
Date:

このように JQMDP は data-dp- で始まるカスタム属性を利用し簡単に動的なページを作成できる。
